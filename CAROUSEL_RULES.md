# IG Carousel 內容與版型規則

- 2026-07-23、2026-07-24 已由使用者鎖定，不再回寫或改動。
- 九天是一個品牌／行業循環；九天使用九個不同的 Template Family。
- 每組九張應形成一條閱讀故事線：封面 Hook → 建立情境 → 解釋 → 關鍵細節 → 視覺停頓 → 比較／選擇 → 證明／幕後 → 收束 → 品牌 CTA。
- 不要求每張同時具備 Logo、Kicker、Headline、Body、Mood、CTA 與 Hashtag。只有當頁敘事需要的欄位才出現；Logo、CTA、Hashtag 原則上集中在封面或收尾。
- 同組保持品牌色、字體與視覺語言一致，但九張構圖及資訊密度要有節奏變化。
- 每張仍需獨立生成為一張 4:5 圖卡，不可一次輸出九宮格；若 Template Family 本身使用模組格或 contact-sheet 語言，允許在單張圖卡內使用。
- 生成圖是 Canva 重建參考；文字、照片框與裝飾元素需要能拆成可編輯圖層。
